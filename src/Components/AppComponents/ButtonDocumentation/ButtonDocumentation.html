Slice Button

To build a simple Slice Button you only need to use the "slice.build" method, just like that you have a Slice Button that you can customize with a value prop. The Button component automatically applies default values and provides enhanced debugging capabilities through Static Props.

Customization

You can also customize your Slice Button specifying its background color with "customColor.button" and its font color with "customColor.label". Static Props ensure that the customColor property is properly typed as an object and defaults to null when not specified.

onClickCallback

At last but not least you can add a function to your button with the prop "onClickCallback". The Static Props system types this callback as a function and defaults to null, with automatic validation in development mode.

Static Props Integration

The Button component leverages Static Props for improved developer experience:

  • Automatic defaults: All props have sensible defaults applied automatically
  • Type validation: Development-time warnings for incorrect prop types
  • Enhanced debugging: Right-click any button to see detailed prop information
  • Self-documenting: Props are clearly defined with types and defaults

Button Static Props Configuration:

static props = {
   value: { type: 'string', default: 'Button', required: false },
   onClickCallback: { type: 'function', default: null },
   customColor: { type: 'object', default: null },
   icon: { type: 'object', default: null }
};
