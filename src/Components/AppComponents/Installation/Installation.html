Installation Guide

Getting started with Slice.js is easy! Follow these simple steps to create your first Slice.js project:

Step 1: Initialize your project

First, create a new directory for your project and initialize it with npm:

npm init
Terminal
Follow the prompts to set up your package.json file. You can press Enter to accept the default values or customize as needed.

Step 2: Install Slice.js CLI

Next, install the Slice.js CLI tool which will help you create components and manage your project:

npm i slicejs-cli
Terminal
added X packages, and audited Y packages in Zs

Step 3: Initialize Slice.js

Initialize the Slice.js folder structure in your project:

npm run slice:init
Terminal
Creating folder structure... Slice framework initialized successfully!

This command creates the necessary folder structure for your Slice.js project, including directories for components, styles, and themes.

Step 4: Start the Server

Finally, start the development server to see your Slice.js application in action:

npm run slice:start
Terminal
Server is running on port 3001, http://localhost:3001

Next Steps

Congratulations! You've successfully set up your Slice.js project. Here's what you can do next:

  • Explore the documentation to learn about components
  • Create your first component with npm run slice:create
  • Check out the built-in Visual components library
  • Build your application by adding routes and services
