Slice Styles

Slice.js provides global styles through the sliceStyles.css file, which establishes consistent base styling across your application. These styles include CSS variables, resets, and utility classes.

CSS Variables

Core CSS variables are defined in the :root element and are available throughout your application.

Base Body Styles

The framework applies minimal base styling to the body element.

Caret/Dropdown Styles

The framework provides styles for dropdown carets used in various components.

Custom Scrollbar

Slice.js includes global custom scrollbar styling for all elements.
